## Why Accordion Pro?

The default YOOtheme accordion is limited. Accordion Pro gives you full control over layout, content, and interaction — without writing a single line of code in the Builder.

---

## Features

- **Custom Toggle Icons** — Replace default chevrons with any UIkit icon or custom SVG
- **Extra Content Blocks** — Add additional text, buttons, or media inside each accordion item
- **Column Layout** — Display accordion content in multi-column grids
- **Modal Integration** — Open detailed content in modals directly from accordion items
- **Badge Support** — Add labels/badges to accordion item headers
- **Description List Layout** — Alternative layout mode for structured data presentation
- **Hover Effects** — Customizable hover states for interactive accordion headers
- **Fully Responsive** — Breakpoint-aware settings for every layout option

---

## Built for YOOtheme Pro Builder

Accordion Pro integrates natively into the YOOtheme Pro Builder interface. All settings are available through the familiar Builder UI — no shortcodes, no custom code required.

**Works with:**
- YOOtheme Pro 4.x
- WordPress 6.x
- PHP 8.0+
- UIkit 3
